
Diagrama IHC (Protótipo) - Domínio Guia Estudantil

Este documento identifica o evento de interação (ação do usuário) para acessar o "Guia Estudantil" para o seguinte caso de uso :
Baixar Guia do Estudante (UC_Guia_3)

A interação ocorre diretamente na Tela Principal do "Minha UFG".
